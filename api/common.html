common.split(target: str, split_words: str) → list¶

組み込みsplit関数の拡張関数。 正規表現を使用しないため高速処理が可能。

Parameters:
  • target (str) – 対象文字列。
  • splitlist (str) – 区切り文字。
Return type:

list
Returns:

区切り文字によって分割された文字列のリスト。

 
>>> split('test//sp"rit%st$ring', '/"%$')
>>> ['test', 'sp', 'rit', 'st', 'ring']
>>> ''.join(split('test//sp"rit%st$ring', '/"%$'))
>>> testspritstring
